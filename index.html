About This Site

This website is built using modern web technologies to simulate the frame-based layout that was popular in earlier web development. The layout includes:

  • Banner Frame: Contains the site title and branding
  • Navigation Frame: Houses the table of contents with links
  • Content Frame: Displays the main content based on navigation selection

Use the navigation menu on the left to explore different sections:

  • Schedule: Weekly schedule displayed in tabular format

Features

This presentation demonstrates:

  • Frame-like layout using CSS Grid
  • Interactive navigation with JavaScript
  • Responsive design principles
  • Modern styling with hover effects
